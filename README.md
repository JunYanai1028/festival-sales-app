# 学園祭店舗向け売上管理アプリ

## 概要
学園祭やイベントで飲食店を出店する学生団体向けに、商品別の売上や販売数を管理できるWebアプリです。

## 作成背景
大学の学園祭で飲食店を出店した際、販売状況を感覚的に判断する場面が多く、どの商品が売れているのか、どの時間帯に販売数が伸びているのかを把握しづらいという課題がありました。

また、チームで店舗運営を行う中で、売上や在庫の状況を共有しにくく、販売戦略の改善につなげるまでに時間がかかる点も課題だと感じました。

そこで、販売状況を簡単に入力し、売上合計や商品別ランキングを確認できるアプリを作成しようと考えました。

## 想定ユーザー
- 学園祭で飲食店を出店する学生団体
- イベントで物販を行うチーム
- 販売状況を簡単に共有したい小規模チーム

## 解決したい課題
- 商品ごとの売上状況が把握しづらい
- 売れ筋商品が分かりにくい
- 販売状況の共有が遅れる
- 改善施策を感覚で決めてしまう

## 主な機能
- 商品名・価格の登録
- 販売数の入力
- 売上合計の自動計算
- 商品別売上ランキングの表示
- localStorageによるデータ保存

## 使用技術
- HTML
- CSS
- JavaScript
- GitHub Pages

## 工夫した点
実際の販売中にスマートフォンで使う場面を想定し、入力項目をできるだけ少なくしました。
また、売上合計や商品別ランキングをすぐに確認できるようにし、販売状況を見ながら改善判断をしやすい構成にしました。
